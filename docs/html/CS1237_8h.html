especially to read it in the background with hardware and timer interrupts,
while the main loop is running and other actions are in progress.

Detailed Description

This is a library for the CS1237-IC to convert the analog readings of a diferential signal in 24bit digital numbers

The Chip communicates through a 2-wire SPI interface SCLK, DRDY/DOUT, two GPIO-pins are required to interface with the CS1237-IC.

Written by Julian Weidinger for an University Project.
